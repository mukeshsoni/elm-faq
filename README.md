#### How to get started?
```
npm i elm -g
mkdir learnelm && cd learnelm
elm-package install
touch LearnElm.elm
# go to http://guide.elm-lang.org/ and copy some example in LearnElm.elm
elm-make LearnElm.elm
open index.html
```
